Listagem de requisitos do projeto Futebol Inglês:

OK, 1. Método [GET] para retornar dados de todos os clubes;
OK, 2. Método [GET] para retornar apenas os dados de um único clube a partir do nome;
OK, 3. Método [GET] para retornar apenas os dados de um único clube a partir do id;
4. Método [POST] para criar um novo clube;
5. Método [POST] para criar um novo elenco dentro de um clube existente;
6. Método [POST] para incluir uma nova cor de uniforme dentro do array de cores de um clube;
7. Método [PUT] para substituir todas as informações de um clube por outro;
8. Método [PATCH] para alterar apenas o treinador de um clube existente;
9. Método [PATCH] para alterar apenas o estádio de um clube existente;
10. Método [PATCH] para alterar apenas um jogar do elenco de um determinado clube;
11. Método [DELETE] para excluir toda uma equipe;
